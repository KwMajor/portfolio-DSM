## Portfólio — DSM (Fatec São José dos Campos)

Projeto pessoal que reúne páginas estáticas com exemplos de projetos, formação e tecnologias utilizadas durante o curso de Desenvolvimento de Sistemas (1º DSM) na Fatec São José dos Campos.

**Orientador:** Jean Carlos

### Tecnologias
- HTML
- CSS
- Bootstrap
- JavaScript

### Estrutura do repositório
- `index.html` — página inicial
- `projetos.html` — seção de projetos (cards padronizados)
- `formacao.html` — formação acadêmica
- `tecnologias.html` — lista de tecnologias com ícones
- `css/style.css` — estilos do site
- `imagens/` — imagens e ícones usados no site
